What technologies were used to build this application?

This application was built using only HTML, CSS, and Vanilla JavaScript without any external libraries or frameworks. The layout uses CSS Flexbox and Grid for responsive design.

How does the sortable list work?

The sortable list uses JavaScript's Array.sort() method to alphabetically sort the names. When you click the "Sort Alphabetically" button, JavaScript collects all list items, sorts them, and then updates the DOM with the sorted order. No page refresh is needed as it's all handled client-side.

Is the form validation done client-side or server-side?

The form validation is done client-side using both HTML5 validation attributes (required, pattern, type) and JavaScript validation. This ensures that data is validated before submission for a better user experience. In a real application, this would typically be paired with server-side validation as well.

How is the contact form submission time calculated?

The contact form tracks the time from when a user first interacts with any field until they successfully submit the form. This is done using JavaScript's Date object to capture timestamps and calculate the difference when the form is submitted.

Is this accordion using JavaScript?

No, this accordion is built using only HTML and CSS. It leverages the <details> and <summary> HTML elements, which provide native browser support for expanding and collapsing content without requiring JavaScript.
